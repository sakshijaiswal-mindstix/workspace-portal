This repository contains the Workspace Portal, organized using a pnpm-based monorepo.
It includes the main application along with shared UI component packages and a fully configured Storybook setup.

## Getting Started
1. Clone the repository
git clone https://github.com/sakshijaiswal-mindstix/workspace-portal
cd workspace-portal

2. Install dependencies
pnpm install

3. Run the main application
pnpm run dev

4. Start Storybook
pnpm storybook
